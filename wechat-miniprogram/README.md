# Ultimate Tic-Tac-Toe 微信小程序 MVP

这是仓库中 Web 版 Ultimate Tic-Tac-Toe 的微信小程序单机版 MVP。它放在独立的 `wechat-miniprogram/` 目录中，不影响根目录 Web 版、GitHub Pages 部署、Supabase 远程对战和现有测试。

## 如何打开

1. 打开微信开发者工具。
2. 选择导入项目。
3. 项目目录选择 `wechat-miniprogram/`。
4. 如果没有正式 AppID，可以选择测试号或使用 `touristappid`。
5. 编译后进入 `pages/game/game` 页面。

## 当前支持

- 本地双人模式。
- 人机模式，玩家为 `X`，电脑为 `O`。
- 普通 / 困难 / 专家 AI。
- 计分板。
- 规则说明弹窗。
- 完整落子历史，默认显示最近几步，可展开 / 收起。
- 撤销。
- 重新开始。
- 重置比分。
- 本地存储保存比分、游戏模式和 AI 难度。

## 规则同步

小程序版复制并整理了 Web 版当前核心规则：

- 小棋盘被赢下后，只要还有空格，仍然可以继续落子。
- 已归属小棋盘的归属不会被后续落子改变。
- 在已归属小棋盘里落子仍然会控制对手下一步去哪个小棋盘。
- 只有目标小棋盘已满时，玩家才可以自由选择其他未满小棋盘。
- 大棋盘胜负仍然只看小棋盘归属。
- 所有小棋盘都下满且没有大棋盘赢家时为平局。

## 暂不支持

- 远程联机。
- 微信登录。
- 排行榜。
- 分享房间。
- 云开发 / CloudBase。
- Supabase 远程同步。

## 说明

Web 版仍然在仓库根目录维护。小程序版第一阶段只做单机可玩版本；后续如果要做联机，建议优先评估微信云开发 / CloudBase 的实时数据库能力。
