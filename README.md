# 终极井字棋 Ultimate Tic-Tac-Toe

一个可在浏览器中运行的 Ultimate Tic-Tac-Toe 小游戏。项目使用原生 HTML、CSS 和 JavaScript 实现，支持本地双人对战、人机对战、AI 难度切换、计分板和比分持久化，适合作为前端小游戏作品集项目展示。

## 游戏规则

- 大棋盘由 9 个小井字棋盘组成，排列为 3 × 3。
- 每个小棋盘内部也是 3 × 3。
- 玩家在某个小棋盘的某个格子落子后，对手下一步必须去对应位置的小棋盘。
- 如果目标小棋盘已经被占领或下满，对手可以选择任意未结束的小棋盘。
- 玩家在小棋盘中三连后，该小棋盘归属该玩家。
- 谁先在大棋盘上占领三个连成一线的小棋盘，谁获胜。
- 如果所有小棋盘都结束且无人完成大棋盘三连，则为平局。

## 功能特性

- 本地双人对战：`X` 和 `O` 轮流落子。
- 人机对战：玩家为 `X`，电脑为 `O`，玩家先手。
- AI 难度：
  - 普通：Ultimate 规则启发式评分。
  - 困难：浅层 Alpha-Beta 搜索 + 启发式评分。
- 清晰显示当前回合、强制落子区域、自由选择原因和游戏结果。
- 当前可落子小棋盘高亮。
- 非法点击有友好提示。
- AI 思考时棋盘进入禁用状态，防止连续点击打乱回合。
- 小棋盘胜利和大棋盘胜利高亮。
- 计分板统计 `X` 胜、`O` 胜和平局。
- `localStorage` 保存比分，刷新页面后仍保留。
- 单局重新开始不会清空比分，重置比分按钮才会清空。
- 响应式布局，适配桌面端和手机端。

## AI 说明

本项目是 Ultimate Tic-Tac-Toe，不是普通 3 × 3 井字棋。AI 不会把普通井字棋的完整 Minimax 直接套到 81 个格子上。

普通 AI 使用启发式评分：

- 优先立即赢下大棋盘。
- 阻止玩家立即赢下大棋盘。
- 评估当前小棋盘的占领价值。
- 评估大棋盘三连威胁。
- 评估落子后会把对手送到哪个小棋盘，避免送到危险区域。
- 中心格和角落格会获得额外位置价值。

困难 AI 使用浅层 Alpha-Beta 搜索：

- 搜索深度为 3 层。
- 候选落子会先按战术价值排序。
- 到达深度上限或游戏结束时，使用同一套 Ultimate 启发式局面评分。
- 搜索带有约 950ms 的时间保护，避免页面卡住。
- 困难 AI 更强，但不保证不败。

## 本地运行

```bash

```

```txt
http://localhost:3000
```

## 测试

```bash
npm test
```

也可以单独做语法检查：

```bash
node --check src/game.js
node --check src/app.js
```

## 构建

```bash
npm run build
```

构建结果会输出到 `dist/`。

## 部署

这是一个零依赖静态站，不是 Vite 项目。

推荐部署方式：

- **GitHub Pages**：可以直接部署仓库根目录，或运行 `npm run build` 后部署 `dist/` 目录。不需要配置 Vite `base`。
- **Vercel**：可以按静态站导入仓库。构建命令填写 `npm run build`，输出目录填写 `dist`。也可以不构建，直接以根目录静态文件方式部署。

