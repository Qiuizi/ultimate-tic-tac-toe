# 终极井字棋

一个适合作品集展示的 Ultimate Tic-Tac-Toe 小游戏。它保留了经典终极井字棋的嵌套规则：玩家在小棋盘中的落子位置，会决定对手下一步必须进入的大棋盘区域。

## 功能

- 9 个小井字棋盘组成一个 3 × 3 大棋盘
- 两名玩家 `X` / `O` 本地轮流对战
- 支持人机对战，玩家使用 `X`，电脑使用 `O`，玩家先手
- 电脑会在玩家落子后短暂思考并自动落子
- 自动限制下一步可落子的区域
- 自动判断小棋盘占领、大棋盘胜利和平局
- 游戏结束后禁止继续落子
- 当前回合、强制区域、已占领棋盘数量清晰展示
- 胜利小棋盘和最终三连区域高亮
- 单局重新开始、悔棋、比分统计、清空比分
- 使用 `localStorage` 保存比分，刷新页面后仍然保留
- 桌面端和手机端响应式布局
- 无外部依赖，结构简单，适合初学者阅读

## 游戏模式

- **两人对战**：本地两名玩家轮流操作 `X` 和 `O`。
- **人机对战**：玩家固定为 `X`，电脑固定为 `O`，玩家先手。切换模式会重置当前棋盘，但不会清空比分。

## AI 策略

电脑不会使用复杂搜索算法，而是按以下优先级选择一个当前合法落子：

1. 如果电脑可以直接赢下整局，优先获胜。
2. 如果玩家下一步可能赢下整局，优先阻挡。
3. 如果电脑可以赢下当前小棋盘，优先占领。
4. 如果玩家即将赢下当前小棋盘，优先阻挡。
5. 如果合法区域内中心格可下，优先下中心。
6. 如果角落格可下，优先下角落。
7. 如果以上都不满足，从剩余合法格子中随机选择。

## 启动方式


然后打开：

```txt
http://localhost:3000
```

## 规则说明

1. 第一步可以在任意小棋盘的任意空格落子。
2. 玩家点击的小格位置，会决定对手下一步必须进入的大棋盘区域。
3. 如果目标区域已经被占领或下满，对手可以自由选择任意未结束的小棋盘。
4. 在某个小棋盘中三连即可占领该小棋盘。
5. 在大棋盘中占领的三个小棋盘连成横、竖或斜线，即获得整局胜利。
6. 如果所有小棋盘都结束且无人完成大棋盘三连，则为平局。
