# 终极井字棋

一个适合作品集展示的 Ultimate Tic-Tac-Toe 小游戏。它保留了经典终极井字棋的嵌套规则：玩家在小棋盘中的落子位置，会决定对手下一步必须进入的大棋盘区域。

## 功能

- 9 个小井字棋盘组成一个 3 × 3 大棋盘
- 两名玩家 `X` / `O` 本地轮流对战
- 支持人机对战，玩家使用 `X`，电脑使用 `O`，玩家先手
- 人机对战支持普通 / 困难两档 AI 难度
- 电脑会在玩家落子后短暂思考并自动落子
- 自动限制下一步可落子的区域
- 自动判断小棋盘占领、大棋盘胜利和平局
- 游戏结束后禁止继续落子
- 当前回合、强制区域、已占领棋盘数量清晰展示
- 胜利小棋盘和最终三连区域高亮
- 单局重新开始、悔棋、比分统计、清空比分
- 使用 `localStorage` 保存比分，刷新页面后仍然保留
- 桌面端和手机端响应式布局
- 无外部依赖，结构简单，适合初学者阅读

## 游戏模式

- **两人对战**：本地两名玩家轮流操作 `X` 和 `O`。
- **人机对战**：玩家固定为 `X`，电脑固定为 `O`，玩家先手。切换模式会重置当前棋盘，但不会清空比分。

## AI 策略

人机对战中玩家固定为 `X`，电脑固定为 `O`。由于 Ultimate Tic-Tac-Toe 不是普通 3 × 3 井字棋，AI 不使用普通井字棋 Minimax 直接套到 81 个格子上。

普通难度使用面向终极井字棋规则的启发式评分：

1. 先检查电脑是否能立即赢下大棋盘。
2. 再检查玩家是否下一步能赢下大棋盘，并优先阻挡。
3. 评估这一步是否能占领当前小棋盘，以及是否能形成大棋盘三连威胁。
4. 评估落子位置会把对手送到哪个小棋盘，避免把对手送去可以立即获胜的小棋盘。
5. 在合法落子中优先选择大棋盘收益高、小棋盘收益高、送板风险低的位置。

困难难度使用浅层 Alpha-Beta 搜索：

- 搜索深度为 3 层。
- 候选落子会先按立即赢大棋盘、阻止大棋盘胜利、赢小棋盘、阻止小棋盘胜利、送板风险、中心和角落价值排序。
- 到达深度上限或游戏结束时，使用同一套 Ultimate 启发式局面评分。
- 搜索带有约 950ms 的时间保护，避免浏览器卡顿。

完整穷举 Ultimate Tic-Tac-Toe 的全局状态空间非常大。困难 AI 更强，但不保证不败；当前优先保证规则正确、交互稳定和响应速度。

## 规则说明

1. 第一步可以在任意小棋盘的任意空格落子。
2. 玩家点击的小格位置，会决定对手下一步必须进入的大棋盘区域。
3. 如果目标区域已经被占领或下满，对手可以自由选择任意未结束的小棋盘。
4. 在某个小棋盘中三连即可占领该小棋盘。
5. 在大棋盘中占领的三个小棋盘连成横、竖或斜线，即获得整局胜利。
6. 如果所有小棋盘都结束且无人完成大棋盘三连，则为平局。
